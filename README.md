# 04-Blinkit-Analysis-SQL-Project

# 🛒 Blinkit E-Commerce Business Analytics Using SQL

## 📖 Project Overview

This project demonstrates how SQL can be used to analyze the operations of an e-commerce grocery delivery business similar to Blinkit. Using a relational database in MySQL, the project explores sales performance, customer behavior, marketing effectiveness, delivery operations, inventory management, and customer satisfaction.

The objective is to transform transactional and operational data into meaningful business insights that support strategic decision-making, improve operational efficiency, and enhance customer experience.

---

# 🎯 Business Objectives

This project aims to answer key business questions across multiple business functions, including:

- Which products generate the highest revenue?
- Which customers contribute the most revenue?
- Which product categories perform best?
- How efficient are delivery operations?
- Which marketing campaigns generate the highest return?
- How satisfied are customers with their purchases?
- Which products experience the highest inventory losses?
- How loyal are customers?
- Which customer segments have the highest retention?
